# API Docs (Module Reference)

This folder documents functions and modules in `src/` in an API-style format.

- `data.md` — dataset formats, prompt builders, validators
- `model.md` — model/tokenizer loading, LoRA config
- `sft.md` — SFT training orchestration
- `rl_grpo.md` — GRPO training orchestration and reward interface
- `eval.md` — evaluation pipeline and metrics
